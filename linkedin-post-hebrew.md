# פוסט לינקדאין - הכירו את מורדכי

**כותרת: הכירו את מורדכי - פלטפורמת הסוכנים הרב-משתמשית שלי בקוד פתוח 🤖**

בניתי משהו מרגש, ואני רוצה לשתף אותכם.

**מורדכי (Mordecai)** היא פלטפורמת סוכני AI מוכנה לייצור עם תמיכה רב-משתמשית, שרצה בצורה מאובטחת על תשתית הענן של AWS. זה לא עוד בוט - זו פלטפורמה שלמה לבניית תהליכי עבודה אוטומטיים וחכמים.

---

## 🏗️ הארכיטקטורה בשורה אחת

![ארכיטקטורת מורדכי](mordecai-architecture.png)

**בנוי על AWS Strands Agents SDK ו- AWS AgentCore**, מורדכי רץ בצורה מאובטחת בענן עם:
- **תורי SQS נפרדים לכל משתמש** לעיבוד אסינכרוני מבודד
- **AgentCore Memory** לידע מתמיד לאורך כל הסשנים
- **שילוב MCP (Model Context Protocol)** לכלים חיצוניים ניתנים להרחבה
- **שירותי FastAPI** עם תמיכה ב-webhooks

## ✨ מה מורדכי יכול לעשות

![תכונות מורדכי](mordecai-features.png)

### מקרים לשימוש בעולם האמיתי:
- **מודיעין דוא"ל** 📧 - סורק את התיבה שלכם ומודיע כשמגיעות הודעות חשובות
- **ניהול משימות** 📋 - יוצר וסוקר משימות ב-Jira עם לוח kanban מובנה
- **תזכורות חכמות** ⏰ - מערכת תזכורות אינטליגנטית עם תזמון cron
- **סיכומים יומיים** 📊 - מרכז נתוני שימוש ב-AWS, פעילות דוא"ל ותובנות
- **בסיס ידע** 🎓 - יוצר הערות מסרטוני יוטיוב ומנהל ידע
- **עוד המון דברים** 🔌 - מערכת skills ניתנת להרחבה עם התקנה אוטומטית ובידוד בין משתמשים
